# 基础编程课程网站

汕头大学长江新闻与传播学院《基础编程（AI 编程与 Python 基础）》课程网站源码仓库。

线上地址：

- https://basic-programming.aisbest.eu.cc/

## 项目简介

这是一个纯静态课程网站，用于集中展示《基础编程》课程的核心信息，包括：

- 课程特色与教学理念
- 16 周课程进度
- 考核方式与评分结构
- 环境配置说明
- AI 编程工具使用建议
- 期末项目要求与选题方向
- 常见问题与学习资源

网站面向零基础学生，内容设计强调“课程说明可直接阅读、教学安排可直接查询、作业与项目要求可直接执行”。

## 技术栈

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages / 自定义域名部署

## 仓库结构

```text
.
├── CNAME

```

## 本地预览

这个项目不依赖构建工具，直接打开 `index.html` 即可查看页面。

如果需要以本地服务方式预览，可以在仓库根目录运行：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 主要文件说明

### `index.html`

页面主体内容，包含：

- Hero 区域
- 课程特色
- 周次安排
- 考核方式
- 环境配置
- AI 工具指南
- 期末项目
- 学习资源
- FAQ

### `index.css`

站点视觉样式文件，包含：

- 深色 / 浅色主题变量
- 响应式布局
- 卡片、表格、导航、按钮等组件样式
- 动效与打印样式

### `main.js`

页面交互逻辑，主要包括：

- 明暗主题切换
- 导航栏滚动状态
- 移动端菜单
- 周课程卡片展开收起
- FAQ 折叠交互
- 滚动渐显效果
- 当前教学周自动高亮

## 维护说明

如果每学期需要更新课程内容，通常只需要修改以下文件：

- 更新课程文案与链接：`index.html`
- 调整视觉样式：`index.css`
- 修改学期开始日期或交互行为：`main.js`

其中，当前教学周的自动高亮由 `main.js` 中的 `SEMESTER_START` 控制：

```js
SEMESTER_START: '2026-02-23'
```

新学期开课后请同步修改为新的周一日期。

## 部署说明

该项目适合直接部署到 GitHub Pages。

如果仓库已启用 Pages：

1. 将源码推送到默认分支
2. 在 GitHub Pages 设置中选择对应分支与根目录
3. 保留 `CNAME` 文件以绑定自定义域名
