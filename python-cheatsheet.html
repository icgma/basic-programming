✂️ 字符串

文本处理是新闻传播专业的核心技能

📐 创建与索引

s = "Hello Python" s[0] → 'H' # 第1个字符 s[-1] → 'n' # 最后一个 s[0:5] → 'Hello' # 切片 s[6:] → 'Python' # 第7个到末尾 len(s) → 12 # 长度

🔧 常用方法

s.upper() → "HELLO PYTHON" s.lower() → "hello python" s.strip() → 去除首尾空格 s.split(",") → 按逗号分割为列表 ",".join(list) → 列表合并为字符串 s.replace("a","b")→ 替换 s.find("Py") → 6 (返回位置) s.count("l") → 2 (统计次数)

💬 f-string 格式化（推荐！）

name = "张三" age = 20 f"我叫{name}，今年{age}岁" # 数字格式化 f"{3.14159:.2f}" → "3.14" f"{156789:,}" → "156,789" f"{0.052:.1%}" → "5.2%"

📋 列表

有序、可变的数据容器

➕ 增

lst.append(x) # 末尾添加 lst.insert(i, x) # 指定位置插入 lst.extend([a,b]) # 批量添加

➖ 删

lst.remove(x) # 按值删除 lst.pop(i) # 按索引删除(返回值) lst.pop() # 删除最后一个 del lst[i] # 按索引删除

🔍 查

x in lst # 是否存在 → True/False lst.index(x) # 查找索引 lst.count(x) # 统计次数 len(lst) # 长度

📊 排序与统计

lst.sort() # 原地升序排序 lst.sort(reverse=True) # 原地降序 sorted(lst) # 返回新列表 lst.sort(key=len) # 按长度排序 sum(lst) / max(lst) / min(lst)

✨ 列表推导式

# 基础 [x*2 for x in range(5)] → [0, 2, 4, 6, 8] # 带条件 [x for x in lst if x > 10] # 字符串处理 [t.strip() for t in titles]

⚠️ 常见陷阱

# sort() 返回 None！ wrong = [3,1,2].sort() # None! # 正确：先.sort()再用, 或用 sorted() # 索引从0开始，最大是 len-1 # lst[len(lst)] → IndexError!

🚨 常见报错速查

遇到了就来这里查，90%的初学者错误都在这里

SyntaxError: invalid syntax
语法错误 — 通常是忘了冒号 :、括号不匹配、引号不配对
✅ 检查 if/for/def/while 末尾是否有冒号
IndentationError: unexpected indent
缩进错误 — 代码缩进不对齐，混用了 Tab 和空格
✅ 统一用 4 个空格缩进，IDE 中设置 Tab = 4 空格
NameError: name 'xxx' is not defined
变量未定义 — 变量名拼错，或者在赋值前就使用了
✅ 检查变量名拼写，确保先赋值后使用
TypeError: can only concatenate str to str
类型不匹配 — 字符串和数字直接用 + 拼接
✅ 用 f"文字{变量}" 或 str(数字) 转换
IndexError: list index out of range
索引越界 — 访问了不存在的列表位置
✅ 索引最大值 = len(lst) - 1，用 -1 获取最后一个
KeyError: 'xxx'
字典键不存在 — 用 d["不存在的键"] 访问
✅ 用 d.get("key", 默认值) 安全访问
ValueError: invalid literal for int()
类型转换失败 — 不能把非数字字符串转为 int
✅ 确保字符串是纯数字；有小数点先转 float 再转 int
FileNotFoundError
文件路径错误 — 文件不在当前目录，或路径写错
✅ 用 os.getcwd() 检查当前目录，用绝对路径
ModuleNotFoundError
库未安装 — 没有用 pip 安装第三方库
✅ 在终端运行 pip install 库名（不是在 Python 里！）
AttributeError: 'xxx' has no attribute 'yyy'
方法名拼错 — Python 区分大小写，.Upper() ≠ .upper()
✅ 检查方法名拼写和大小写
